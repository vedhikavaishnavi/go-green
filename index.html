What is Go Green?
The Go Green Initiative was created by an enterprising high school student to help people gain awareness about how even the smallest changes in their lifestyle can make an impact on saving the environment. The more people in our society that make these changes, the more progress we make in saving the environment. This initiative needs more awareness so we realise that if we don't start saving the environment, soon there won't be one for us to live in.
Explore this website more to find out some methods on how to save the environment.
